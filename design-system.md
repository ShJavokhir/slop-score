# SlopScoreDesign System

**Core Philosophy:** Radical simplicity prioritizing content legibility over decoration. The design channels technical documentation aesthetics - Wikipedia meets modern SaaS.

## Typography

**Hierarchy:** Clear three-tier system. Primary headings use substantial size with medium weight. Body text sits at comfortable reading size (likely 16-17px). Monospaced font for technical terms (repo names, code). Secondary text (descriptions, metadata) uses muted gray at reduced size.

**Spacing:** Generous line-height for prose content. Tight letter-spacing on headings. Lists and nested navigation use consistent indentation.

## Color Palette

**Background:** Pure white (#FFFFFF) for main canvas, subtle gray (#F7F7F8) for sidebar/cards.

**Text:** Near-black for primary content, medium gray (#6B7280 range) for secondary information.

**Accents:** Single blue for interactive elements - links, buttons, active states. Appears to be in the #3B82F6 range.

**Borders:** Extremely subtle - almost imperceptible dividers between sections using light gray.

## Layout & Spacing

**Grid:** Homepage uses 3-column card grid with consistent gaps. Repo view employs classic sidebar (240px fixed) + main content + secondary sidebar pattern.

**Whitespace:** Abundant padding within cards and sections. Margins follow 8px base unit system (16px, 24px, 32px, 48px visible).

**Containers:** Content constrained to ~1200px max-width, centered.

## Components

**Cards:** Subtle background, rounded corners (~8px), minimal shadow/border, hover state likely adds subtle shadow.

**Search:** Full-width input with icon, rounded corners, subtle border.

**Navigation:** Text-only links, active state uses bold weight + subtle background.

**Buttons:** Blue fill, white text, medium padding, rounded corners.

**Tables:** Borderless rows, subtle dividers, aligned columns.

**Critical detail:** No visual noise. Icons used sparingly (stars, arrows). Dark mode toggle present but design defaults to clarity in light mode first.
